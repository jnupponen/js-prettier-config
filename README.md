# js-prettier-config

## Install prettier

```
npm install --save-dev prettier
```

## Add config file `prettier.config.js` at project root

```
module.exports = {
    bracketSpacing: true,
    jsxBracketSameLine: false,
    printWidth: 90,
    semi: false,
    singleQuote: false,
    tabWidth: 2,
    trailingComma: "none",
    useTabs: false
}
```
